# 📝 User Registration Form

A modern and responsive front-end registration form focused on delivering a smooth and interactive user experience with real-time validation.

## ✨ Features

* 📱 Fully responsive design (mobile + desktop)
* 🎨 Clean and modern UI
* ✅ Real-time form validation with visual feedback
* 🔐 Password visibility toggle (eye icon)
* 📊 Password strength indicator
* ⚡ Smooth and interactive user experience
* 🔄 Dynamic input fields based on user selection

## 🚀 Tech Stack

* Frontend Development (UI/UX + Client-side Logic)
